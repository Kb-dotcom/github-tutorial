FCOR 599 Capstone Project:

Brief Overview:
This project explores the use of Unmanned Aerial Vehicles (UAVs) equipped with multispectral sensors and GIS-based techniques to identify and segment vineyard rows for precision viticulture. By combining remote sensing data with spatial analysis, I aim to identify vineyard row IDs and spatial statistics.


Objectives:
To determine the NDRE values of vines after discriminating from weeds and soil
To identify vineyard rows using image analysis.
To calculate the zonal statistics of each vine after segmentation.


Methods:
Image acquisition
Image Analysis
Row segmentation
